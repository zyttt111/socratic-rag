## 功能模块待办

### Tier 1：Agent 核心能力（最加分）

- [ ] **Multi-Agent 辩论** (`src/agents/debate/`)
  - 辩论调度器（回合制、自由辩论）
  - 回合管理 & 发言顺序控制
  - 裁判评分 & 观点收敛机制
  - 分歧点自动识别 & 聚焦讨论

- [ ] **LangGraph 工作流** (`src/agents/graph/`)
  - Socratic RAG → 辩论 → 图谱增强多步推理链
  - 状态管理（对话历史、上下文窗口）
  - 条件路由（问题类型 → 不同链路）
  - Checkpoint / Human-in-the-loop

- [ ] **Agent 工具集** (`src/agents/tools/`)
  - 文献搜索工具（书名/章节/关键词）
  - 概念查询工具（定义、关联概念）
  - 图谱检索工具（哲学家关联、影响链）
  - 论文/外部文献检索工具

- [ ] **哲学家 Persona 运行时** (`src/agents/philosophers/`)
  - Persona 注入（YAML → System Prompt）
  - 风格控制（语气、论证方式、引用偏好）
  - 辩论发言生成（带哲学家视角）

### Tier 2：工程质量（面试必问）

- [ ] **测试体系**
  - pytest-asyncio 异步测试
  - 覆盖率目标 70%+
  - LLM Mock（不调真实 API 的单元测试）
  - RAG 检索精度回归测试

- [ ] **CI/CD**
  - GitHub Actions：lint → test → type-check
  - PR 自动检查（ruff、mypy、pytest）
  - 自动构建 Docker 镜像

- [ ] **可观测性**
  - structlog 结构化日志
  - LLM 调用延迟追踪
  - Token 消耗统计
  - RAG 检索命中率监控

- [ ] **错误处理 & 韧性**
  - 指数退避重试
  - 熔断器（连续失败 → 降级）
  - 优雅降级（LLM 不可用时返回缓存结果）

- [ ] **API 完善**
  - 分页 & 限流
  - OpenAPI 文档完整度
  - 请求/响应 Schema 规范化

- [ ] **部署**
  - 多阶段 Dockerfile
  - Health check
  - 生产/开发配置分离

### Tier 3：差异化亮点

- [ ] **RAG 评估升级**
  - A/B 测试框架（不同 chunk 策略对比）
  - 回归测试集管理
  - 自动评估报告生成

- [ ] **GraphRAG 增强推理**
  - 概念关联推理（概念 → 相关概念 → 哲学家）
  - 影响链分析（康德 → 黑格尔 → 马克思）
  - 图中路径可视化

- [ ] **流式辩论**
  - SSE / WebSocket 实时推送
  - 辩论过程可视化

- [ ] **自适应学习路径**
  - 基于 MemoryManager 的用户模型
  - 难度递进推荐
  - 知识点掌握度追踪

- [ ] **多语言文献**
  - 中英德三语原文对齐
  - 跨语言概念检索
